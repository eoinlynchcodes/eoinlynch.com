Visebox.com

Visebox was a website that let you find, schedule and speak to someone that has done what you aspire to do for business, sport or education. We had a platform that handled payment, scheduling discussion, inbuilt video call software and more. We pivoted to connect prospective college students to students of the college courses they were considering enrolling in to gain an understanding of the reality of being a student in a course.

The idea of Visebox came to me after being mentored by Svein Tuft as a bike racer and realising the value someone can get learning from another person with experience, wisdom and knowledge. A friend, Jeremiah Scully became a co-founder and was instrumental in getting the product online. After working on Visebox, I knew that I needed to be able to code so enrolled in Lambda School.

Lambda School

Lambda School is a 9+ month accelerated Full-Stack Software Engineering & Computer Science program with an immersive (full-time, 40+ hours/week) hands-on curriculum based in San Francisco, backed by Stripe, Alphabet, and Y Combinator. I finished it August 2020.

  • Learned to use HTML, CSS, JavaScript ES6, React, Context API, Redux, Jest, Express, MongoDB, PostgreSQL, Python, Knex, Git and more.
  • Built and deployed REST APIs writing production-ready code using Node/Express and Heroku with MongoDB, AWS and PostgreSQL.
  • Worked under pressure to pass 3 hour code challenges each Friday.
  • Experienced building several products with a team using Agile and the Git workflow.
  • Pair programmed frequently, and worked remotely with EU, US based teams.
  • Learned Computer Science fundamentals and made projects associated with Data Structures, Algorithms, Algorithm Efficiency, Hash Tables, Graphs and Computer Architecture.
